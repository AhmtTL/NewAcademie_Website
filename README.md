# Newyork Empire Academy Website

Laravel Backend + React Forntend + SQLite Database Project

## Project Structure

- **backend/** → Laravel(PHP + SQLite)

- **frontend/** → React


### Backend Installation 

cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve

For detailed backend instructions, click → nyea-backend/README.md

cd frontend
npm install
npm run dev

For detailed backend instructions, click → nyea-frontend/README.md
